## 三个标签页功能

### 1. 上下文结果
显示每个关键字命中的详细上下文：
- 关键字名称
- 行号和时间戳
- 前 N 行（上文）
- 命中行（高亮显示）
- 后 N 行（下文）

### 2. 关键字共现
显示多个关键字在同一区域内共同出现的代码块：
- 共现的关键字列表
- 行范围
- 完整的代码块内容
- 帮助发现关联问题

### 3. 时间线视图
按时间顺序展示关键字事件：
- 时间戳
- 关键字
- 行号
- 行内容
- 适合追踪问题发生的时间序列

## 典型使用场景

### 场景 1：错误排查
```
输入关键字：
ERROR
Exception
Failed

结果：快速定位所有错误，查看错误上下文和堆栈信息
```

### 场景 2：性能分析
```
输入关键字：
timeout
slow
latency

结果：通过共现分析，发现哪些超时与慢查询相关
```

### 场景 3：用户行为追踪
```
输入关键字：
user_id=12345
login
purchase
logout

结果：通过时间线视图，追踪用户完整操作路径
```
